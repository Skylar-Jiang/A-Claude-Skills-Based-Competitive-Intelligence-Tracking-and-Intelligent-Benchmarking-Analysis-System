# 接口说明（兼容入口）

本文件保留旧文件名，避免历史链接失效。当前权威、与代码及 OpenAPI 一致的接口文档为：

- [`API接口文档.md`](API接口文档.md)
- Swagger：服务启动后的 `GET /docs`

## 当前正式验收顺序

1. `GET /`：健康检查。
2. `GET /sources`：查看启用的真实公开数据源。
3. `POST /collect/run?force=true&use_llm_filter=false`：真实采集。
4. `GET /records`：查看清洗后的可追溯记录。
5. `POST /rag/rebuild`：使用 `BAAI/bge-small-zh-v1.5` 重建 Chroma。
6. `GET /rag/search?query=黄瓜价格涨跌`：返回原文、来源名称、真实链接与发布时间。
7. `POST /analysis/run`：`mode=real` 运行正式 LangChain `RunnableSequence` 分析。
8. `POST /report/generate`、`GET /reports`：生成及查询报告。

旧 `*_manual.csv` 与 `manual.local` 仅为明确标注的课堂 sample，不属于正式数据，不进入正式 RAG。正式链路只读取 `data/processed/intelligence_records.csv` 中来源、链接、发布时间和采集时间均完整的公开记录。真实模式失败返回明确 502/503，不会静默切换 Mock。
